# Frontend (Next.js) — Mien‑Termin

Dieses Verzeichnis enthält das Next.js Frontend. Workflows und Komponenten werden auf Deutsch dokumentiert.

Kurze Anleitung:

1. Installiere Node.js (empfohlen v18+) und npm/yarn.
2. Wechsle in `frontend/` und führe `npm install` aus.
3. Starte das Dev‑Server: `npm run dev`.

API‑URL (lokal): `NEXT_PUBLIC_API_URL=http://127.0.0.1:8000/api`
