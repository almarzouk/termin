# Contributing

Please follow these simple rules when contributing to this repository:

- Branches: use `feature/<short>-description` for new features, `hotfix/<issue>` for urgent fixes.
- Base branch: `main` (protected). Create PRs to merge feature branches.
- Commits: use Conventional Commits (e.g., `feat:`, `fix:`, `chore:`).
- PR template: describe changes, how to run locally, and migrations required.
- Reviews: require at least one approving review and passing CI checks before merge.
