# Contributing (Auf Deutsch)

Bitte folgende Regeln beim Beitragen zu diesem Repository beachten:

- Branches: Verwende `feature/<kurz>-beschreibung` für neue Features und `hotfix/<issue>` für dringende Fixes.
- Hauptzweig: `main` (geschützt). Erstelle PRs, um Feature‑Branches zu mergen.
- Commits: Nutze Conventional Commits (z.B. `feat:`, `fix:`, `chore:`).
- PR‑Vorlage: Beschreibe die Änderungen, wie lokal getestet wird, und notwendige Migrationsschritte.
- Reviews: Mindestens eine Genehmigung und erfolgreiche CI‑Checks erforderlich vor dem Merge.
